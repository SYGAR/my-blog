    清除浮动的四种方法

    SYGAR 2/2/2023 HTML&CSS

    页面布局当中，在无法确定子元素的高度（height）时，我们无法给父级标签一个固定的高度（height），我们想要的是，由子元素的高度去控制父元素的高度。 所以，当子元素设置了浮动（float）属性之后，父元素的高度没有进行设置，这样就会导致父元素的高度塌陷(原本的height被重置为0)，这就涉及到了清除浮动的重要性

    # 1.额外标签法

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .app{
      background-color:green;
    }
    .app1{
      background-color: red;
      float: left;
    }
    .app2{
      background-color: yellow;
      float: left;
    }
  </style>
</head>
<body>
  <div class="app">
    <div class="app1">手机</div>
    <div class="app2">电脑</div>
    <!-- 添加一个空的标签 -->
    <div class="clear" style="clear: both;"></div> 
  </div>
</body>
</html>

    # 2.父元素添加 overflow: hidden

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .app{
      overflow: hidden;
      background-color:green;
    }
    .app1{
      background-color: red;
      float: left;
    }
    .app2{
      background-color: yellow;
      float: left;
    }
  </style>
</head>
<body>
  <div class="app">
    <div class="app1">手机</div>
    <div class="app2">电脑</div>
  </div>
</body>
</html>

    # 3.:after 伪元素法，为父元素清除浮动

    <!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .clearfix:after {
      content: "";
      display: block;
      height: 0;
      clear: both;
      visibility: hidden;
    }

    /*IE6、7专有*/
    .clearfix {
      *zoom: 1;
    }

    .app {
      background-color: green;
    }

    .app1 {
      background-color: red;
      float: left;
    }

    .app2 {
      background-color: yellow;
      float: left;
    }
  </style>
</head>

<body>
  <div class="app clearfix">
    <div class="app1">手机</div>
    <div class="app2">电脑</div>
  </div>
</body>

</html>

    # 4.:after和:before双伪元素清除浮动

    <!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .clearfix:after,
    .clearfix:before {
      content: '';
      display: table;
    }

    .clearfix:after {
      clear: both;
    }

    .clearfix {
      /* 为 IE6、IE7浏览器设置的清除浮动 */
      *zoom: 1;
    }

    .app {
      background-color: green;
    }

    .app1 {
      background-color: red;
      float: left;
    }

    .app2 {
      background-color: yellow;
      float: left;
    }
  </style>
</head>

<body>
  <div class="app clearfix">
    <div class="app1">手机</div>
    <div class="app2">电脑</div>
  </div>
</body>

</html>
